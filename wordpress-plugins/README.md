## 🔐 Security Focus

Two of the three current plugins are
specifically security focused — addressing
the most common WordPress attack vectors:

**Default admin URL exposure**
The default /wp-admin URL is the first
target for brute force attacks.
Changing it reduces attack surface immediately.

**Username enumeration**
Exposed admin usernames allow attackers
to focus brute force on valid accounts.
Hiding credentials blocks this vector.

These plugins came directly from real
client security incidents — not theory.

## 💡 Why Custom Plugins

Many developers install third party plugins
for problems that can be solved with
50 to 150 lines of clean custom code.

Custom plugins mean:
- No bloat or unnecessary features
- No third party update dependency
- Full control over functionality
- No licensing costs
- Better performance
