# Interactive Event Invite (GIF Gala RSVP)

An interactive event invitation page built with **HTML, CSS, and JavaScript** that allows users to RSVP for an event and receive instant confirmation feedback.

## 📌 Project Overview

The **GIF Gala RSVP** project is a fun and responsive event invitation webpage where visitors can:

* Read event details
* Submit their RSVP
* Indicate attendance status
* Receive an instant confirmation message without page reloads

This project demonstrates fundamental front-end development concepts including form handling, DOM manipulation, event listeners, and responsive UI design.

## 🚀 Features

* Clean and modern event invitation layout
* RSVP form with email input
* Attendance selection dropdown
* Client-side form processing using JavaScript
* Dynamic confirmation messages
* Responsive design for different screen sizes
* Simple and beginner-friendly code structure

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
