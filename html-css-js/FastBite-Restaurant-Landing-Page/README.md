# 🍔 FastBite Restaurant Landing Page

A modern and responsive fast-food restaurant landing page built using **HTML, CSS, JavaScript, and Bootstrap 5**.

This project demonstrates a professional restaurant website featuring a responsive navigation menu, interactive food menu, shopping cart functionality, order section, and mobile-friendly design.

---

## 🚀 Live Features

### 🏠 Home Section
- Modern hero banner
- Call-to-action buttons
- Responsive layout
- Fast-food themed branding

### 🍟 Our Menu
- Product card grid layout
- Food images
- Product descriptions
- Price display
- Add to Cart button
- Order Now button

### 🛒 Shopping Cart
- Dynamic cart counter
- Add items to cart
- Remove items from cart
- LocalStorage support
- Cart data persists after page refresh

### 📦 Load More Products
- Initial product listing
- Load more button
- Additional products loaded dynamically

### 📋 Order Section
- Customer order form
- Name field
- Phone field
- Delivery address field
- Order confirmation message

### 📱 Responsive Design
- Mobile-friendly layout
- Bootstrap responsive grid
- Mobile navigation menu
- Optimized for desktop, tablet, and mobile devices
