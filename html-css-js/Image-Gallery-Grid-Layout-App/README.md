# Impact Gallery - Responsive Image Showcase

A responsive image gallery project built using HTML, CSS, and JavaScript.  
This project demonstrates layout structuring, DOM manipulation, and responsive UI design without any frameworks or external libraries.

---

## Project Structure

image-gallery-showcase/
│
├── index.html
├── style.css
├── script.js
└── README.md

---

## Features

- Responsive image grid using CSS Grid
- Image lightbox (click to enlarge preview)
- Featured image carousel using thumbnails
- Mobile responsive navigation menu
- Keyboard support (ESC to close lightbox)
- Click outside modal to close lightbox
- Clean modern UI using custom CSS (no frameworks)

---

## Sections Overview

### Header
- Logo on left
- Navigation links (Home, About, Gallery)
- Mobile hamburger menu

### Hero Section
- Title and short description
- Gradient background styling

### Image Grid
- 3x3 responsive grid layout
- Clickable images for lightbox view

### Lightbox
- Fullscreen image preview
- Close button (X)
- ESC key support
- Click outside to close

### Featured Carousel
- Large main image display
- Thumbnail row below
- Click thumbnail to change main image
- Active thumbnail highlighting

### Footer
- Developer branding
- Contact information included

## Technologies Used

- HTML5
- CSS3 (Grid, Flexbox, Media Queries)
- JavaScript (DOM Manipulation, Events)

---

## Responsive Behavior

- Desktop: 3-column grid layout
- Tablet: 2-column grid layout
- Mobile: 1-column layout
- Navigation collapses into hamburger menu

---

## JavaScript Features

- Lightbox open/close functionality
- Keyboard event handling (ESC key)
- Image carousel thumbnail switching
- Mobile navigation toggle

---

## Notes

- No external libraries or frameworks used
- No CDN dependencies (pure vanilla implementation)
- Images loaded using placeholder service (picsum.photos)
