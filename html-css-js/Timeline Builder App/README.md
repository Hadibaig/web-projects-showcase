# Timeline Builder

A lightweight Timeline Builder application built with HTML, CSS, and JavaScript.

## Features

- Add timeline events
- Delete timeline events
- Real-time timeline preview
- Responsive design
- Clean and modern interface
- No frameworks required

## Technologies

- HTML5
- CSS3
- JavaScript (ES6)
