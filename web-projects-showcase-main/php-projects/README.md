## 📐 Development Standards

Every project in this folder follows:

- PSR coding standards for PHP
- Prepared statements for all database queries
- Input sanitization and output escaping
- Separation of logic and presentation
- Error handling and logging
- Clean and commented code structure

---

## 🔐 Security Practices Applied

- SQL injection prevention via PDO prepared statements
- XSS prevention via proper output escaping
- CSRF protection on all forms
- Password hashing using PHP password_hash
- Sensitive data kept in environment variables
- No credentials hardcoded in source files

---

## 💡 Why Pure PHP

Understanding PHP at its core — without
a framework — makes you a better developer
because you understand exactly what
frameworks are abstracting away from you.

These projects demonstrate that understanding
before moving into framework and CMS based
development.
