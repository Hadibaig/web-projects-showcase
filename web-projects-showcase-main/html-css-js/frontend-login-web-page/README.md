# Frontend Login Showcase

A modern responsive login system built using HTML, CSS, JavaScript, and Bootstrap.

This project demonstrates a professional frontend login interface with modern UI components, responsive layouts, dark mode support, animated backgrounds, and JavaScript-based authentication logic.

---

## Features

* Responsive Design
* Bootstrap 5 Layout
* Modern Login UI
* Show / Hide Password
* Remember Me Functionality
* Forgot Password Page
* Dark / Light Mode Toggle
* Animated Background Particles
* User Avatar / Logo Support
* Session Storage Authentication
* Local Storage Authentication
* Mobile Friendly Design

---

## Technologies

* HTML5
* CSS3
* JavaScript (ES6)
* Bootstrap 5
* tsParticles
