## 📐 Theme Development Approach

Every theme and customization in this
collection follows these principles:

**Child themes over core edits**
All customizations use child themes —
never modifying parent theme files directly.
This ensures update safety and maintainability.

**Hook based customizations**
Wherever possible, customizations use
WordPress action and filter hooks —
the correct and future proof approach.

**Performance first**
Every theme is built with page speed
and Core Web Vitals in mind from the start —
not optimized as an afterthought.

**Mobile first**
All designs start from mobile breakpoints
and scale up — not the other way around.
