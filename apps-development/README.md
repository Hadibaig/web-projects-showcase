## Applications

### ButtonCraft Generator

A visual button generator that enables developers and designers to create customizable buttons and instantly export production-ready code for use in websites and applications.

**Highlights**

* Live visual editor
* Production-ready code generation
* Accessibility-first output
* Built-in animations
* Icon support
* Responsive preview

---

### Geoscope Content Analyzer

An AI-focused content analysis tool designed to help creators optimize content for both traditional search engines and AI-powered search experiences through Generative Engine Optimization (GEO).

**Highlights**

* Traditional SEO analysis
* GEO (AI Search) analysis
* Readability evaluation
* Keyword analysis
* Content quality scoring
* Real-time recommendations
